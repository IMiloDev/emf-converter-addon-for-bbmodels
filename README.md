# EMF Converter Addon  
By IMilo 

> Plugin for Blockbench  
> Converts Blockbench animations (visual keyframes) into mathematical expressions compatible with [EMF (Entity Model Features)](https://modrinth.com/mod/entity-model-features), generating a ready-to-use Minecraft resource pack.

For mandatory requirements and limitations, see: [Important.md](Important.md)

## 🇬🇧 English

### What does it do?

1. Reads a `.bbmodel` file containing your animations.  
2. Lets you assign each animation a behavior type (idle, walk, custom).  
3. Converts keyframes into trigonometric expressions usable by EMF at runtime.  
4. Generates `player.jem` and `player_slim.jem`, preparing a resource pack structure.

----

### Installation

1. Download the .zip file
2. Unzip it
3. Open Blockbench
4. Click File, then Addons
5. Add the addon from your desktop
6. Enjoy
---

### Usage

1. Create an empty folder on your computer  
2. Open your `.bbmodel` in Blockbench  
3. Run the EMF Converter Addon  
4. Select animation types (idle / walk / custom)  
5. Export into the folder you created  
6. Compress it into a `.zip` and place it in your resource packs folder
